Loan debt - Patron payment

Title: Patron loan history acceptance flow Patron->DDB CMS: Open Loan status debts. DDB CMS->FBS: Request Patron debt (Fees). FBS->DDB CMS: Return list of debts. DDB CMS->Patron: Display list of debts. Patron->DDB CMS: Pay debt. DDB CMS->DIBS: Redirect to DIBS payment DIBS->Patron: Prompt for payment. Patron->DIBS: Pays debt. DIBS->DDB CMS: Return information of payment Note right of DDB CMS: Add to DIBS reports: \n (/admin/config/payment/ding_dibs/reports: \n ORDER ID | TRANSACTION ID | DATE | USER | \n TEXT | AMOUNT | CLEARED PAID | CAPTURED) DDB CMS->Economy System: Post amount, Order ID and Transaction ID. Note right of Economy System: This integration does not exist yet. \n See https://platform.dandigbib.org/issues/3877 for details. DDB CMS->Patron: Display updated list of debts.
