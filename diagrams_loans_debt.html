Loan debt - Patron payment

Title: Patron loan history acceptance flow. Patron->DDB CMS: Open Loan status debts. DDB CMS->FBS: Request Patron debt (Fees). FBS->DDB CMS: Return list of debts. DDB CMS->Patron: Display list of debts. Patron->DDB CMS: Pay debt. DDB CMS->DIBS: Redirect to DIBS payment DIBS->Patron: Prompt for payment. Patron->DIBS: Pays debt. DIBS->DDB CMS: Return information of payment DDB CMS->DDB CMS: Add to DIBS reports (/admin/config/payment/ding_dibs/reports: ORDER ID | TRANSACTION ID | DATE | USER | TEXT | AMOUNT | CLEARED PAID | CAPTURED) DDB CMS->Patron: Display updated list of debts.
